# Project: Template

各プロジェクトにファイルのテンプレートを提供するプロジェクトです。

## 使い方

ファイルをコピーし、ファイルに書いてあるように編集します。

## 開発

### 環境構築

1. Nix を使う方法

- Nix と direnv をインストールします。
- このディレクトリで、 `direnv allow` を実行します。

2. それ以外

- 以下のパッケージをインストールしてください。
  - Bun
  - Biome
  - Just
